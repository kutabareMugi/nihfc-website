# NIHFC Investment Platform

A modern, luxury fintech landing page built with React, TypeScript, and Tailwind CSS. Features a sophisticated "Quiet Luxury" design system with dark mode aesthetics, animated components, and interactive wealth simulation tools.

## Features

- **Premium Design System** - "Onyx & Champagne" color palette with glass morphism effects
- **Interactive Wealth Simulator** - Visualize investment growth over 7 years with real-time calculations
- **Budget Archetypes** - Personalized financial guidance based on user's financial position
- **Mobile App Showcase** - Dynamic phone mockup with animated portfolio tracking
- **Future Opportunities** - Horizontal scroll gallery of emerging investment sectors
- **Responsive Design** - Mobile-first approach with smooth animations and transitions

## Tech Stack

- **React 18** with TypeScript
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Recharts** for data visualization
- **Radix UI** for accessible components
- **Vite** for fast development and building

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Key Components

- **HeroSection** - Animated hero with particle background
- **VaultSection** - Portfolio diversification visualization
- **WealthSimulator** - Interactive investment calculator with charts
- **BudgetArchetypes** - Tabbed interface for financial archetypes
- **AppShowcase** - Mobile app mockup with live data
- **FutureHorizons** - Scrollable opportunity cards

## Design Philosophy

The project follows a "Quiet Luxury" aesthetic with:
- Dark mode by default (Onyx background)
- Premium accents (Champagne, Electric Blue, Emerald)
- Subtle animations and micro-interactions
- Glass morphism effects
- Premium typography (Space Grotesk, Playfair Display)
